Projet : Visualisation des commandes – Northwind

1. Description
Ce projet présente l’analyse des commandes livrées et non livrées à l’aide de Power BI
et de visualisations Python intégrées.  
Les données proviennent de la base Northwind, extraites depuis deux sources :
- Northwind dans SQL Server
- Northwind dans Microsoft Access

2. Outils utilisés
- Power BI
- Python (Pandas, Matplotlib)
- Overleaf (LaTeX)
- CapCut (vidéo)

3. Exécution des scripts
Les scripts Python sont utilisés dans Power BI via le visuel Python.
Il suffit de :
- Ouvrir le fichier Northwind.pbix
- Sélectionner un visuel Python
- Coller le code correspondant
- Cocher les colonnes nécessaires dans le panneau Données

4. Reproduction des résultats
- Charger les données depuis SQL Server et Access
- Vérifier le modèle relationnel
- Utiliser les mesures existantes
- Générer les visualisations

5. Sources des données
- Northwind SQL Server : tables de faits et dimensions
- Northwind Access : tables complémentaires et informations clients
